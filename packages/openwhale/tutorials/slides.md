# Presentation Creation

Generate PowerPoint and PDF presentations.

## Capabilities

- Create slide decks from prompts
- Generate with images and diagrams
- Export to PPTX or PDF
- Multiple layout options

## Basic Commands

### Quick deck
```
Create a 5-slide presentation about machine learning basics
```

### Detailed outline
```
Create a presentation with:
- Title: Q4 Sales Review
- Slide 1: Overview
- Slide 2: Key Metrics
- Slide 3: Regional Breakdown
- Slide 4: Challenges
- Slide 5: Q1 Goals
```

## Examples

### Business pitch
```
Create a startup pitch deck for a food delivery app with 10 slides
```

### Educational
```
Make a presentation teaching Python basics for beginners
```

### Report
```
Turn this quarterly report into a presentation with key highlights
```

### Technical
```
Create slides explaining our system architecture with diagrams
```

## Formatting Options

### With images
```
Create a travel presentation about Japan with images for each destination
```

### Charts
```
Create slides showing our growth with charts from this data: [data]
```

### Themes
```
Make a professional dark-themed presentation about cybersecurity
```

## Tips

- Slides include speaker notes by default
- Images are generated or sourced automatically
- Bullet points are auto-formatted
- Consistent styling throughout

## Output Formats

### PowerPoint
```
Create a PPTX presentation about...
```

### PDF
```
Create a PDF slide deck about...
```

## Output Location

Files are saved to `.openwhale-temp/` by default.

Specify a location:
```
Create the presentation and save it to ~/Presentations/pitch.pptx
```
