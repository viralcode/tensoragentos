Features

Everything you'd expect from an OS, reimagined around AI.

🧠 AI-First Interface

The entire desktop is an AI agent. Chat to control your computer — run commands, manage files, install software, all through natural language.

🖥️ Native Desktop Shell

Custom Qt6 QML desktop with window management, dock, top bar, and system tray. Runs on Wayland via the Cage compositor.

⚡ Deep Kernel Access

50+ OS tools give the AI native access to processes, services, filesystem, network, hardware, users, packages, and kernel parameters.

🔌 Any AI Provider

Works with Anthropic, OpenAI, Google, DeepSeek, Ollama, and more. Switch providers on the fly from the settings panel.

🤖 Multi-Agent

Create specialized agents, coordinate multi-agent workflows, and delegate tasks across agents with shared context and memory.

📡 Multi-Channel

Connect WhatsApp, Telegram, Discord, and iMessage. Your OS agent responds across all messaging platforms.

💾 Persistent Memory

Remembers conversations, preferences, and context across sessions. Built-in SQLite-backed memory and vector search.

🔧 Extensible Skills

GitHub, Weather, Notion, Spotify, Trello, and more. Create custom skills with markdown files — hot-reloaded at runtime.

🌐 Browser Automation

Built-in headless Chromium for web browsing, scraping, and automation — all controllable through the AI chat.

Get Started

Boot your own AI operating system in under 5 minutes.

1

Install QEMU

On macOS with Homebrew:

brew install qemu
2

Download the base image

Debian 12 ARM64 cloud image (for Apple Silicon):

curl -L -o vm/debian-generic.qcow2 "https://cloud.debian.org/images/cloud/bookworm/latest/debian-12-generic-arm64.qcow2"
3

Clone and launch

Everything is automatic — cloud-init installs Node.js, Qt6, OpenWhale, and the desktop shell.

git clone https://github.com/viralcode/tensoragentos.git && cd tensoragentos && python3 vm/launch-ainux.py
4

Wait ~5 minutes

First boot installs everything automatically then reboots into the desktop GUI.

5

Log in and talk

Login with admin / admin. Type into the chat bar. Your AI agent is ready.

SSH: ssh ainux@localhost -p 2222 (password: ainux)
