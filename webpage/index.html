Get Started

Boot your own AI operating system in under 5 minutes.

1

Prerequisites

You need QEMU and Python 3 installed. On macOS with Homebrew:

brew install qemu python3
2

Download Debian Base Image

Download the Debian 12 ARM64 cloud image (for Apple Silicon):

curl -L -o vm/debian-generic.qcow2 "https://cloud.debian.org/images/cloud/bookworm/latest/debian-12-generic-arm64.qcow2"
3

Clone & Launch

Clone the repo and run the launcher. It builds and boots the OS automatically:

git clone https://github.com/viralcode/tensoragentos.git && cd tensoragentos && python3 vm/launch-ainux.py
4

Wait for First Boot

Cloud-init installs everything automatically (Node.js, Qt6, OpenWhale, etc). Takes ~3-5 minutes. The VM reboots into the GUI on its own.

5

Log In & Talk

Login with admin / admin. Type into the chat bar at the bottom of the desktop. Your AI agent is ready.

SSH: ssh ainux@localhost -p 2222 (password: ainux)
